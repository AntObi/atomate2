CalculationOutput

CalculationOutput¶

pydantic model atomate2.vasp.schemas.calculation.CalculationOutput[source]¶

Document defining VASP calculation outputs.

field energy: float = None¶

The final total DFT energy for the calculation

field energy_per_atom: float = None¶

The final DFT energy per atom for the calculation

field structure: Structure = None¶

The final structure from the calculation

field efermi: float = None¶

The Fermi level from the calculation in eV

field is_metal: bool = None¶

Whether the system is metallic

field bandgap: float = None¶

The band gap from the calculation in eV

field cbm: float = None¶

The conduction band minimum in eV (if system is not metallic)

field vbm: float = None¶

The valence band maximum in eV (if system is not metallic)

field is_gap_direct: bool = None¶

Whether the band gap is direct

field direct_gap: float = None¶

Direct band gap in eV (if system is not metallic)

field transition: str = None¶

Band gap transition given by CBM and VBM k-points

field mag_density: float = None¶

The magnetization density, defined as total_mag/volume (units of A^-3)

field epsilon_static: Tuple[Tuple[float, float, float], Tuple[float, float, float], Tuple[float, float, float]] = None¶

The high-frequency dielectric constant

field epsilon_static_wolfe: Tuple[Tuple[float, float, float], Tuple[float, float, float], Tuple[float, float, float]] = None¶

The high-frequency dielectric constant w/o local field effects

field epsilon_ionic: Tuple[Tuple[float, float, float], Tuple[float, float, float], Tuple[float, float, float]] = None¶

The ionic part of the dielectric constant

field frequency_dependent_dielectric: FrequencyDependentDielectric = None¶

Frequency-dependent dielectric information from an LOPTICS calculation

field ionic_steps: List[Dict[str, Any]] = None¶

Energy, forces, and structure for each ionic step

field locpot: Dict[int, List[float]] = None¶

Average of the local potential along the crystal axes

field outcar: Dict[str, Any] = None¶

Information extracted from the OUTCAR file

field force_constants: List[List[Tuple[Tuple[float, float, float], Tuple[float, float, float], Tuple[float, float, float]]]] = None¶

Force constants between every pair of atoms in the structure

field normalmode_frequencies: List[float] = None¶

Frequencies in THz of the normal modes at Gamma

field normalmode_eigenvals: List[float] = None¶

Normal mode eigenvalues of phonon modes at Gamma. Note the unit changed between VASP 5 and 6.

field normalmode_eigenvecs: List[List[Tuple[float, float, float]]] = None¶

Normal mode eigenvectors of phonon modes at Gamma

field elph_displaced_structures: ElectronPhononDisplacedStructures = None¶

Electron-phonon displaced structures, generated by setting PHON_LMC = True.

field dos_properties: Dict[str, Dict[str, Dict[str, float]]] = None¶

Element- and orbital-projected band properties (in eV) for the DOS. All properties are with respect to the Fermi level.

field run_stats: RunStatistics = None¶

Summary of runtime statistics for this calculation

classmethod from_vasp_outputs(vasprun: Vasprun, outcar: Outcar, locpot: Optional[Locpot] = None, elph_poscars: Optional[List[Path]] = None) → CalculationOutput[source]¶

Create a VASP output document from VASP outputs.

Parameters
vasprun

A Vasprun object.

outcar

An Outcar object.

locpot

A Locpot object.

Returns
The VASP calculation output document.
